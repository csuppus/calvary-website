This file describes the directories and files in this website:

app.yaml - main configuration file for GAE
index.yaml - database indexing configuration for GAE
robots.txt - robots file to control how search engines index the site
base/ - contains modules for shared components between church and school
church/ - modules specific to the church
school/ - modules specific to the school
plugins/ - modules that can be plugged into many pages
css/ - css for the site
scripts/ - javascript modules
media/ - static media files for the site, like banners, link images, etc
templates/ - all html template files, used to generate html for the site
